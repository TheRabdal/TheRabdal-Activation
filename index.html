Cara Mengubah .txt menjadi .bat

Jika file yang diunduh berformat .txt, Anda perlu mengubah namanya (rename) menjadi .bat agar dapat dijalankan. Berikut langkah-langkahnya:

  1. Buka File Explorer dan pilih menu View (Tampilan) di bagian atas.
  2. Centang kotak File name extensions (Ekstensi nama file).
  3. Klik kanan pada file yang diunduh, lalu pilih Rename (Ubah nama).
  4. Ganti akhiran .txt menjadi .bat, tekan Enter lalu pilih Yes.
